Advanced Certificate in Project Management (ACPM)
သင္တန္းအေျကာင္းအရာ ဘယ္လိုလူေတြတက္သင့္သလဲ?

သင္တန္းအေျကာင္းအရာ

Singapore ႏုိင္ငံ Cedar Academy မွ Advanced Project Management တြင္ သတ္မွတ္ခ်ိန္ကာလနွင့္ ေငြေၾကးအစီအစဥ္အတိုင္း လုပ္ငန္းတိုးတက္ရာ တိုးတက္ေၾကာင္း အေျပာင္းအလဲမ်ားကို အရည္အေသြးျပည့္မွီစြာ စီမံခန္႔ခြဲျခင္းကို ေခၚပါသည္။ Project တစ္ခုသည္ ပံုမွန္လုပ္ငန္း လည္ပတ္မွဳ လုပ္ငန္းစဥ္မ်ားနွင့္ မတူညီပါ။ ဥပမာအားျဖင့္ အေဆာက္အဦးအသစ္ တည္ေဆာက္ျခင္း၊ ပစၥည္း/၀န္ေဆာင္မွဳ အသစ္မ်ား တီထြင္ထုတ္လုပ္ျခင္း၊သတင္းအခ်က္အလက္ စနစ္ထူေထာင္ျခင္း နွင့္ လက္ေတြ႔အသံုးခ်ျခင္း၊ ေဈးကြက္အသစ္ခ်ဲ႕ထြင္ျခင္း၊ လူစြမ္းအား အရင္းအျမစ္ တိုးတက္ေရး စီမံခ်က္အသစ္ ခ်မွတ္ျခင္း နွင့္အေကာင္အထည္ေဖာ္ျခင္း စသည္တို႔မွာ အဖြဲ႔အစည္းမ်ားအတြင္း ေတြ႔ရေလ႔ရွိေသာ Project မ်ား ျဖစ္ပါသည္။ Project တစ္ခုတြင္ လုပ္ငန္း စတင္ခ်ိန္နွင့္အဆံုးသတ္ခ်ိန္ရွိျပီး Project ေအာင္ျမင္မွဳအတြက္ လိုအပ္ေသာ လူ၊ ေငြေၾကး၊ သတင္းအခ်က္၊ စက္ပစၥည္းစသည့္ အရင္းအျမစ္မ်ား သီးသန္႔ သတ္မွတ္ထားရေလ႔ရွိပါသည္။ Project တစ္ခုသည္ အဖြဲ႔အစည္းတစ္ရပ္၏ ထူးျခားေသာ အေျပာင္းအလဲ ျဖစ္ျပီး ျပတ္သားရွင္းလင္းေသာ ရလဒ္တစ္ခုအတြက္ ရပ္တည္ေနေသာ လုပ္ငန္းစဥ္လည္း ျဖစ္ပါသည္။ Project တစ္ခုေအာင္ျမင္ေရးအတြက္ အဖြဲ႔အစည္းတစ္ခုလံုး၊ ၄င္း၏ HR, Marketing, Finance, အစရွိသည့္ အထူးျပဳဌာနမ်ား အားလံုး ပူးေပါင္းျပီး ေဆာင္ရြက္ရေလ့ရွိပါသည္။

ဘယ္လိုလူေတြတက္သင့္သလဲ?

Project, Finance, Ngos, IT, စီးပြားေရး စီမံခန္႔ခြဲမွဳ နယ္ပယ္မ်ားတြင္ အလုပ္လုပ္ကိုင္ေနၾကသူမ်ား။ -ပုဂၢလိက နွင့္ အစိုးရ လုပ္ငန္းမ်ား တြင္ လုပ္ကိုင္ေနၾကသူမ်ားႏွင္႔ Porject Mangement ကိုစနစ္တက် ေလ႔လာလုိသည့္ လုပ္ငန္းရွင္မ်ား။ စီးပြားေရး ေငြေၾကး ရင္းနွီးျမွဳပ္နွံမွဳကို ရလဒ္ေကာင္းမ်ား ရရွိေအာင္ စနစ္တက်ျပဳလုပ္လိုသည္႔ စီးပြားေရး လုပ္ငန္းရွင္မ်ားနွင့္ ရင္းနွီးျမွဳပ္နွံသူမ်ားနွင့္ အစိုးရအဖြဲ႔အစည္းမ်ား ၏ စီမံကိန္း (Project) မ်ားတြင္ ဘ႑ာေရးနွင့္ ေငြေၾကးကို တာ၀န္ယူေနျပီး စနစ္တက်စီမံခန္႔ခြဲလိုသူမ်ား တက္ေရာက္သင့္ပါသည္။
